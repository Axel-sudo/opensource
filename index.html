01 What's "Open source"?

An open source software is a computer program whose source code is distributed under a license which allows anyone to read, modify and redistribute it.

02 How can it protect my privacy?

There is no direct correlation between open source and privacy but unlike proprietary softwares, the transparency of the source code found in open source licensed softwares permits us to see what they collect from us and even modify it to remove those snooping parts,

wheras private softwares don’t allow us to do such things, leaving you in the dark where data collecting is concerned.
