What's "Open source"?

An open source software is a computer program whose source code is distributed under a license allowing anyone to read, modify and redistribute this software

02

How it can protect my privacy?

There is no direct link between open source and privacy but unlike proprietary software, the transparency of the source present in open source licensed softwares permits us to see what they collect from us or even modify it to remove those snooping parts,
wheras the private ones don’t allow us to do such things, thus leaving you in the dark where data collecting is concerned.
